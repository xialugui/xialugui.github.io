# 土龙从入门到精通

土龙即湮灭之卡利斯，是目前游戏中难度最高的副本BOSS。本手册将从零到一，逐步帮助各位玩家攻略土龙。
## 绪论
### 土龙介绍
#### 阶段划分
现阶段土龙一共有24个阶段，在13阶段后土龙会变身银龙，19阶段后冰龙进入最后的狂暴形态（尾巴翅膀变黄），25阶段后土龙进入终极形态黄金龙（困难模式存在，本文只针对24阶段简单土龙）。
#### 主要机制
包括蓄力喷射龙焰、转圈甩尾二连、捡盾牌机制、标记追踪、吼叫机制、狩猎机制等。变身为银龙后，还会新增全屏冲刺、召唤分身等机制（分身机制一、二阶段同样存在）。
### 比雷龙的变化
#### 机制复杂度
雷龙的机制相对较为简单，固定连招接爪击后接甩尾，站在雷龙背后可规避大量攻击。而土龙机制更为复杂，有更多的阶段变化，且部分机制会随着阶段变化而增强。
#### 攻击方式
土龙的攻击方式更加多样化，其甩尾是一个多段、带有浮空判定的甩击，前摇极短，若紧贴其身后，很容易被拍到天上秒杀。此外，土龙的吼叫机制也与雷龙不同，土龙吼叫时会往地上落攻击，很难像在雷龙吼叫时那样安心输出。
#### 防御机制
土龙具有独特的防御机制，当血量掉到一定程度时，会获得高额防御加成，如血量降到 67%以下会获得200%的防御，降到35%以下会增长500%的防御，这使得在攻击土龙时，伤害计算和输出节奏与雷龙有很大不同。
### 难点所在
#### 机制难度
土龙的机制不仅复杂，而且部分机制的执行难度较高。例如，捡盾牌机制要求必须正面朝向龙息，背对吐息则会被打掉盾牌，且盾牌还可能被地上的龙刺炸掉；冰龙的全屏冲刺速度极快，伤害高，需要有良好的走位才能躲避。
#### 输出压力
由于土龙的防御较高，且在不同阶段有防御加成，这就要求在有限的输出窗口内打出足够的伤害，对装备打造、技能连招熟练度以及团队的配合度都有很高的要求。
#### 生存考验
土龙的攻击伤害高，范围大，稍有不慎就可能被秒杀，特别是在一些机制联动时，如标记追踪结束后的龙刺引爆，以及银龙阶段的大范围攻击，都需要时刻保持警惕，注意走位和躲避。

## 阶段
简单土龙分为4个大阶段，每个阶段的机制都上个阶段的加强版。各个阶段的血量肉度估算见下表
| 阶段 | 血量范围（%） | 防御倍率（%）|肉度占比（%）|
|:----:|:----:|:------:|:------:|
| 一 | 89-100 | 0 | 3 |
| 二 | 67-89 | 30 | 8 |
| 三 |  35-67 |  200 | 27 |
| 四 |  0-35 |  500 | 60 |
 
四阶段血量占比达百分之60，这意味着要斩龙，必须熟练所有机制且制定合理的输出与闪避策略。
## 技能详解与应对策略
### 一阶段
#### 抓
![](images/2025-10-27-23-06-36.png)
原地抓，前方三分叉攻击。
- 应对
  + 向龙后方闪避； 
  + 向龙前方闪避；

#### 横扫
![](images/2025-11-03-12-23-59.png)
前方矩形区域横扫，范围较小，。
- 注意：
  + 回首掏很难躲（使用抓头等短无敌技能存在风险），建议上/下方走位；
  + 建议上/下方走位；
- 应对
  + 向龙后方闪避（走位），回首偷输出（三阶段不适合，建议躲避形成肌肉记忆）；
  + 向龙上/下方走位；

#### 扇击
![](images/2025-10-27-23-10-38.png)
原地扫尾，呈扇形。
- 应对
  + 向龙后方闪避； 
  + 向龙前方闪避；

#### 后撤步
![](images/2025-11-03-12-27-49.png)
土龙前方出现无分叉的原地拍击，后续接一个后撤步横扫。
![](images/2025-11-03-12-29-22.png)
- 应对
  + 向龙后方闪避（走位）；
- 注意：
  + 后撤步横扫的攻击范围实际要小些，可以贴着红框输出；
#### 摆尾
![](images/2025-11-03-12-32-05.png)
一阶段的摆尾无特殊规律，随着技能循环出现。
- 应对：
  + 在龙前方；
- 注意：
  + 保持站位，随时调整位置至龙前方，摆尾的首次攻击前摇非常短，被击中后浮空，土龙连招一套秒；
#### 吸吐
![](images/2025-11-03-12-36-05.png)
先吸后吐，秒杀机制。
![](images/2025-11-03-13-28-19.png)
吐息时红框位置是安全输出环境，边缘半个身位无伤害。
- 应对：
  + 在龙后方走位，保持距离偷输出；
  + 待大圆圈填充满之前输出，填充满时远离；
  + 可以使用觉醒来暂时抵消吸附；

### 二阶段
![](images/2025-11-03-13-29-42.png)
龙鳞变黄进入二阶段（7阶段），二阶段机制较一阶段难，但应对方式差异不大。
#### 抓
![](images/2025-11-03-13-42-43.png)
原地抓，前方三分叉攻击，比一阶段范围大。
- 应对
  + 向龙后方闪避（走位），建议闪避后立即调整位置至龙前方（三阶段很大概率接摆尾）；
  + 向龙前方闪避（走位）；
  + 攻击过后50%概率接飞扑![](images/2025-11-03-13-46-16.png)，注意调整走位；
- 注意：
  + 闪避到龙后方可以立即偷一个后摇极短的技能，然后再调整身位（风险大）；
#### 前抓
![](images/2025-11-03-13-36-50.png)
![](images/2025-11-03-13-38-27.png)
土龙向前方进行2次爪击（三阶段变为三次，最后一次回首掏）。
- 应对
  + 向龙后方闪避（走位），回首偷输出；
#### 扇击
应对方式相同。
#### 横扫
![](images/2025-11-03-13-40-03.png)
范围比一阶段大，应对方式相同。

#### 后撤步
![](images/2025-11-03-14-03-22.png)
与一阶段一致，但范围变大，应对方式一样。
- 注意：
  + 后撤步的横扫带有延迟的三次弧斩，避免矩形区域消失立即进入![](images/2025-11-03-14-04-48.png)；
#### 呼啦圈
![](images/2025-11-03-13-53-52.png)
呼啦圈共2次，伤害高。
- 应对
  + 迅速进入呼啦圈中心位置（可以偷一个技能），两次呼啦圈攻击范围不重叠；
  + 第一次建议接无敌技能（后面阶段呼啦圈攻击范围重叠），也可以接闪避，直接闪避（可以选择保留闪避，使用无敌技能）到下一次呼啦圈龙的水平位置，提前调整走位；
  + 两次呼啦圈位置有规律，一次上，那么下一次就在下![](images/2025-11-03-13-59-05.png)，要么在左，要么在右；

- 注意：
  + 偷输出时要注意鳞片爆炸的时机，使用无敌技能的时间要把握得刚刚好；

#### 追击
![](images/2025-11-03-14-05-10.png)
土龙追击目标，持续12.5s。

- 应对
  + 脚下圆圈外安全距离，可安全输出，黄色三角攻击区域进入圆圈建议立刻远离；
  + 黄色三角攻击区域进入圆圈是一些技能的作用范围（怒气），此时应择机使用技能；
  + 使用技能攻击的安全距离为屏幕出现半个龙头；
  + 此时是使用觉醒的较好时机；
  + 红色圆圈扩大到红线时，可以走位躲过，省下闪避（红线扩散到白线为龙小跳追击时间）；
  + 走位至离圈较近位置可偷输出；
- 注意：
  + 黄色三角区域的角范围比看起来要大，在此位置要谨慎；
  + 一些技能会带有隐性位移（鬼斩），谨慎使用；
  + 龙的最后小跳追击有延迟，约0.5s，勿提前走位；

 
![](images/2025-11-03-14-05-54.png)
#### 引鳞
![](images/2025-11-03-13-47-17.png)
发射追踪鳞片，一共发射12片。
- 应对
  + 地图边缘走位，减少鳞片爆炸范围；
  + 尾巴拍地后鳞片爆炸![](images/2025-11-03-13-50-57.png)，此时可以使用无敌技能躲避；
  + 组队时要注意队友得位置，不要让队友被鳞片波及；
- 注意：
  + 偷输出时要注意鳞片爆炸的时机，使用无敌技能的时间要把握得刚刚好；

#### 喷鳞
![](images/2025-11-03-14-00-33.png)
全屏喷，脚下不安全，但危险度低，注意走位，适合输出。
- 注意：
  + 尽量避免被鳞片击中；
#### 捡鳞
土龙完成二阶段变身后马上触发捡鳞机制。
![](images/2025-10-27-23-12-43.png)
掉落4个鳞片，必须一人一个，未捡必死。持续8秒。
- 注意：
  + 被（吐息过程中被地下的鳞片爆炸）击中，鳞片掉落必死![](images/2025-11-03-13-33-07.png)；
  + 背对土龙读条满秒杀（可以短暂背对土龙，调整位置以防被鳞片爆炸击中）；
- 应对
  + 一二阶段的捡鳞机制是偷输出的好机会，因为地面鳞片不多，使用觉醒抵消位移影响；
  + 一些技能如大吸、血剑要距离土龙非常近（一个身位）才适合释放，不然易空；

### 三阶段
土龙13万血进入三阶段（13阶段），颜色变为银色，并立即触发飞斩机制。
![](images/2025-11-03-14-17-27.png)
掉落黄色增加技能颗粒，一颗百分之1。
#### 飞斩
![](images/2025-11-03-14-17-52.png)
- 应对
  + 屏幕视角拉大之前调整位置至水平中央（龙是左右飞斩，左右预留空间增加反应时间）；
  + 屏幕视角拉大的瞬间上下走位；
  + 观察到龙飞离屏幕立刻开始走位![](images/2025-11-03-14-20-48.png)，或在地面出现小横条的瞬间开始走位；
  + 手指建议放在闪避键上，随时准备闪避；
  + 可以使用无双波等减伤技能躲避；
  + 落地等待红圈快满使用无敌技能，或者使用闪避![](images/2025-11-03-14-27-45.png)；
- 注意：
  + 上下走位时，注意不要在两个方向犹豫，犹豫必死；
  + 被击中会浮空，生存概率很小，建议随时闪避；
  + 组队要么站同一条水平线，同时躲避，要么垂直方向分散站开，使用闪避（一共四次斩击，两人都不走位的前提下，每人使用两次闪避，三人组队一人只要1.3次闪避，比走位保险）；
  + 使用风华套/乌龟套死亡概率降低（不会被击飞）；
#### 抓
![](images/2025-11-03-14-49-46.png)
抓击，带三分叉。
- 应对
  + 走位（闪避）到后方，接技能回首掏，立刻调整到龙前方![](images/2025-11-03-14-54-25.png)![](images/2025-11-03-14-54-40.png)![](images/2025-11-03-14-55-03.png)；
- 注意：
  + 爪击后50%接摆尾，50%接飞扑一连（飞扑一连带4分叉）；
#### 前抓
![](images/2025-11-03-14-47-30.png)
2次向前抓击，接回首掏。
![](images/2025-11-03-14-48-20.png)
- 应对
  + 第一次闪避到身后，偷输出；
  + 第二次接着偷；
  + 回首掏要么接无敌技能，要么向前走位，远离攻击范围；

#### 扇斩
![](images/2025-11-03-14-28-56.png)
范围比一阶段大，应对方式相同。
- 注意：
  + 闪避（走位）到龙背后太快的话，会立刻再接一个扇斩，建议接无敌技能（小吸）；
#### 横扫
![](images/2025-11-03-15-01-58.png)
横扫两次，第二次大概率回首掏，应对方式相同。

#### 后撤步
![](images/2025-11-03-14-55-40.png)
应对方式相同。
#### 呼啦圈
![](images/2025-11-03-14-56-15.png)
![](images/2025-11-03-14-56-28.png)
范围变大，两次有重叠，且必接#highlight[摆尾]。
- 应对
  + 走位（闪避）到第一次的内圈再向相反向走位，风险低，大概率远离摆尾区域；
  + 走位（闪避）到第一次的内圈接无敌技能，风险低；
  + 走位（闪避）到第一次的内圈，再闪避到第二次位置，保持和龙一条水平线，且调整到龙前方（龙后方极危险，立刻就有摆尾机制）；
- 注意：
  + 呼啦圈后必接摆尾，宁可不输出也要躲避摆尾；
#### 上下喷
![](images/2025-11-03-14-30-58.png)
跳到中央吼叫1s，脚底出圈，三个分身一个真身。
- 应对
  + 红圈外偷输出；
  + 红圈内使用无敌技能；
  + 使用扎热可以降低观察真身的难度；
  + 实体为真身，可输出；
  + 站在靠红色区域较近的位置进行输出（使用一个闪避可以移动到上下安全区域，太远了一个闪避不够用）![](images/2025-11-03-14-35-22.png)；
  + 在红线边缘可以看到上下龙视野，最多查看一次就能找出真身位置![](images/2025-11-03-14-37-21.png)；
  + 安全位置没有真身时可以引爆鳞片，清理场地；
  + 机制最后龙会跳到中央，此时有1s时间可以偷输出![](images/2025-11-03-14-38-45.png)；
- 注意：
  + 分身时伤害很高，注意躲避![](images/2025-11-03-14-32-52.png)；
  + 此机制过后大概率接吸吐，喷鳞；
#### 吸吐
![](images/2025-11-03-14-40-01.png)
应对方式相同，注意捡起发光黄色颗粒。

#### 摆尾
![](images/2025-11-03-14-41-45.png)
有概率直接使用摆尾，注意将身位保持在龙前方。摆尾分为三次，第一、二次前摇极短，第三次较慢，可偷技能并接无敌技能。
![](images/2025-11-03-14-43-49.png) 
- 应对
  + 龙前方有安全区域，可输出；

#### 飞扑
![](images/2025-11-03-15-03-15.png)
乌鸦坐飞机三次，不带分叉。
- 应对
  + 落下一瞬间可偷输出；
- 注意：
  + 建议将圈保持在中央，边缘位置没有走位空间，极易死亡；

#### 引鳞
![](images/2025-11-03-14-45-25.png)
12片，应对方式相同。
#### 喷鳞
![](images/2025-11-03-14-46-01.png)
此阶段的危险度适中，可以偷技能，不过要接闪避（必要时）。
#### 捡鳞
![](images/2025-11-03-15-07-36.png)
应对方式相同，注意不要贪输出，及时上下调整走位。



### 四阶段
![](images/2025-11-03-15-09-48.png)
7万血（19阶段）进入四阶段，身体部分变黄。并立即出发走栅格机制。
#### 栅格
![](images/2025-11-03-15-11-27.png)
周围向中心爆炸，龙脚底下为安全区域。
- 应对
  + 范围缩小时全力输出；
  + 龙飞天后将鳞片引爆![](images/2025-11-03-15-13-15.png)；
  + 可以使用鳞片击倒效果蹲伏躲过2次栅格机制；
  + 龙落地范围大，可使用闪避、无敌和快速位移![](images/2025-11-03-15-17-13.png)；
- 注意：
  + 距离太远使用闪避；
##### 栅格形式
有五种形式
+ ![](images/2025-11-03-15-13-51.png)，下方左右两个；
+ ![](images/2025-11-03-15-14-21.png)，左/右一个；
+ ![](images/2025-11-03-15-15-00.png)，角落一个；
+ ![](images/2025-11-03-15-15-28.png)，左右边缘一个；
+ ![](images/2025-11-03-15-15-28.png)，斜两个；
不管哪种，将身位调整至中央，减少移动距离。

增加了小横格子斩，![](images/2025-11-03-15-28-47.png)，每个格子带有冲击波，应对方式相同。

应对方式相同。

  + 此时的后撤步横扫是三连击

应对方式相同。
#### 飞斩
![](images/2025-11-03-15-32-42.png)

+ 学会在龙攻击间隙输出是通关的关键；
